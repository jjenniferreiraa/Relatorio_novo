# Relatorio Simcon-Solos-Sal 2025
Repositório para construção do relatório 2025 de prestação de contas do projeto Simcon-Solos-Sal

# Observações:
1 - Trocar o compilador do bibtex para Biber. No Texstudio, abrir as opções, marcar o checkbox avançadas e alterar a seguinte linha

![image](https://github.com/user-attachments/assets/25ffe339-8700-4211-94cc-979c48c62b8e)

2 - Todos os pacotes utilizados na versão anterior e que não precisaram ser aplicados até o momento estão comentados no arquivo pacotes.tex

3 - O arquivo bibliografia.bib foi reaproveitado e corrigido. Antes de inserir uma referência, verifique se ela já não existe e, quando inserir, atentar para caracteres especiais (acentos e cedilhas, principalmente)

4 - Cada novo anexo deve criar uma pasta dentro da pastas *figuras* para guardar as imagens (preferencialmente em PDF)
